# bigben

index.html is a webpage that will display the clock face of Big Ben and, optionally, play ticks and/or the appropriate chimes.

You can access the webpage at [https://heffalump.mit.edu/bigben](https://heffalump.mit.edu/bigben).

Acknowledgements:

* clock face: https://bigbentime.com
* sounds: https://sound-effects.bbcrewind.co.uk/search?q=Big%20Ben
